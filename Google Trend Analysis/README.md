# Global Google Search Trends Dashboard

An interactive Power BI dashboard built to track global Google search trends for job-related keywords such as accountant, data science, and finance analyst. The dashboard highlights rising search terms, monitors search popularity over time, and compares regional interest to reveal patterns in job demand and skill trends.  

This tool allows users to enter any keyword, instantly updating visuals to display top searches, emerging terms, and regional breakdowns. It provides actionable insights for professionals exploring career shifts, businesses monitoring talent needs, and educators tracking emerging skills, demonstrating practical applications of analytics, reporting, and business intelligence.

## Project Overview
This Power BI dashboard tracks global job-related Google search trends and provides interactive insights for strategic decision-making. It demonstrates how business intelligence can reveal patterns in labor demand, skill popularity, and emerging career opportunities.  

**Start Date:** [Insert Date]  
**Purpose:** Monitor trends in job-related searches for business and educational insights  


## Key Features
- **Interactive Keyword Search:** Enter any job-related keyword to dynamically update visuals  
- **Trending Terms:** Highlights rising search terms and emerging skills  
- **Regional Analysis:** Compare search interest by country or region  
- **Popularity Over Time:** Track changes in search volume across months or years  
- **Dynamic Visuals:** Charts, maps, and tables that update instantly  


## Use Cases
- **Professionals:** Explore career shifts or emerging job opportunities  
- **Businesses:** Monitor talent demand for recruitment and skill development  
- **Educators & Institutions:** Track future skills and adapt training programs  
- **Analysts:** Understand global labor market trends and regional preferences  


## Dashboard Structure
- **Keyword Input Panel:** Allows dynamic entry of search terms  
- **Trend Charts:** Display search popularity over time  
- **Regional Heatmaps:** Show top regions for selected keywords  
- **Emerging Terms Table:** Lists rising search terms relevant to the selected keyword  
- **Interactive Filters:** Enable quick exploration by region, category, or time frame
